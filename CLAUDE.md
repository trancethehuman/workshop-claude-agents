# Building AI Agents with Claude Code & Agent SDK

An 8-week hands-on workshop teaching you to build AI agents for data analysis, GTM, marketing, and business automation.

## Key Insight

Claude Code is not just for writing code. It's an agent that can:
- Analyze CSV files and datasets
- Research companies and competitors
- Draft emails and content
- Process documents and reports
- Automate repetitive business tasks

## Repository Structure

- `lessons/` - Session plan MDs (week1-week8) + `images/`
- `labs/` - Lab exercise MDs (16+ labs) + `images/`
- `quizzes/` - Quiz MDs: `answer-key/` and `student/`
- `evals/` - Evaluation materials
- `data/` - Sample datasets for exercises
- `references/` - Supporting documentation
- `scripts/` - Setup and utility scripts
- `admin/` - Non-instructor files (docx, decks, pdfs, diagrams)

## How to Use

When working in this repo, Claude should:
- Be concise and practical
- Explain what it's doing and why
- Show which tools it's using
- Save outputs appropriately

## Preferences

- Use simple, readable solutions
- Explain reasoning before taking action
- Format outputs as markdown tables when appropriate
- Ask clarifying questions when requirements are unclear
